# 1 Поиск
## 1.1 Поиск по имени группы/преподователя

**GET /api/v2/search?q={query}** <a name="search"></a>

**Ответ**

```javascript
    [
        {
            "id":1,
            "name":"АБб-14-1",
            "url":"АБб-14-1",
            "type":"group"
        },
        {
            "id":7,
            "name":"Абрамова Т.В.",
            "url":"Абрамова_ТВ",
            "type":"teacher"
        }
    ]
```
